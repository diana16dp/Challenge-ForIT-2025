# Gestor de Tareas - Aplicación FullStack
Esta aplicación permite organizar tus tareas de una manera fácil y practica. Además de contar con un check para marcar tus tareas finalizadas!

## Ejecución
Clonar el proyecto

```bash
  git clone https://github.com/diana16dp/Challenge-ForIT-2025.git
  cd Challenge-ForIT-2025
```

- Ejecutar primero el backend
```bash
  cd backend
  npm install
  npm run dev
```

- Ejecutar luego el frontend
```bash
  cd frontend
  npm install
  npm run dev
```
